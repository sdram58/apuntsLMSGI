Els llenguatges de marques:{
~%-33.333%Permeten representar informació
~%-33.333%Es basen en marques o etiquetes
~%-33.333%No són llenguatges de programació
=Totes són vàlides
}

L'exemple més representatiu de llenguatges de marques orientat a la presentació d'informació és:{
~%-33.333%GML
~%-33.333%SGML
=HTML
~%-33.333%XML
}

HTML és un llenguatge de marques creat a partir de:{
=SGML
~%-33.333%XML
~%-33.333%XHTML
~%-33.333%SVG
}

Assenyala quina de les opcions següents és una de les característiques fonamentals dels llenguatges de marques:
{

~%-33.333%S'emmagatzemen en formats complexos codificats
~%-33.333%Són rígids

}
