Variables en CSS

A CSS, les propietats personalitzades (també conegudes com a variables) són entitats definides per autors de CSS que contenen valors específics que es poden tornar a utilitzar en un document. S'estableixen mitjançant la notació de propietats personalitzades (per exemple, --main-color: black;) i s'hi accedeixen mitjançant la funció var() (per exemple, color: var (--main-color);).

Llocs web complexos tenen quantitats molt grans de CSS, sovint amb una gran quantitat de valors repetits. Per exemple, el mateix color es pot utilitzar a centenars de llocs diferents, la qual cosa requereix buscar i reemplaçar globalment si necessitem canviar aquest color. Les variables CSS permeten que un valor s'emmagatzemi en un lloc i després es faci referència a diversos altres llocs. Un benefici addicional són els identificadors semàntics. Per exemple --main-text-color és més fàcil d'entendre que #00ff00, especialment si aquest mateix color també es fa servir en un altre context.

Les variables CSS estan subjectes a la cascada, i hereten el valor del pare.

Ús Bàsic

Per declarar propietats personalitzades (variables) fem servir un nom que comenci amb dos guions (--), i un valor que pot ser qualsevol valor vàlid de CSS. Com qualsevol altra propietat, l'escrivim dins d'un set de regles d'estil, així:

element {
  --main-bg-color: brown;
}

Tingueu en compte que el selector que fem servir per a les regles d'estil defineix l'àmbit (scope) en què podrem fer servir la propietat personalitzada (variable). Una bona pràctica comuna és declarar variables a la pseudo-classe :root, i així aplicar-les globalment al document HTML:

:root {
  --main-bg-color: brown;
}

Tanmateix, això no ha de ser sempre així: podrien haver-hi molt bones raons per voler limitar l'àmbit de les teues propietats personalitzades.

Els noms de propietats personalitzades són case sensitive (distingeixen entre majúscules i minúscules) — --my-color serà tractat com una propietat personalitzada diferent de --My-color.

Com hem esmentat anteriorment, per accedir al valor d'una propietat personalitzada fem servir el nom de la propietat dins de la funció var(), en lloc de qualsevol altre valor normal:

element {
  background-color: var(--main-bg-color);
}

Primers passos amb propietats personalitzades (variables)

Comencem amb aquest simple CSS que pinta elements de diferents classes amb el mateix color:

.un {
  color: white;
  background-color: brown;
  margin: 10px;
  width: 50px;
  height: 50px;
  display: inline-block;
}

.dos {
  color: white;
  background-color: black;
  margin: 10px;
  width: 150px;
  height: 70px;
  display: inline-block;
}
.tres {
  color: white;
  background-color: brown;
  margin: 10px;
  width: 75px;
}
.quatre {
  color: white;
  background-color: brown;
  margin: 10px;
  width: 100px;
}

.cinc {
  background-color: brown;
}

Ho aplicarem a aquest HTML:

<div>
  <div class="un">1:</div>
  <div class="dos">2: Text <span class="cinc">5 - més text</span></div>
  <input class="tres" />
  <textarea class="quatre">4: Lorem Ipsum</textarea>
</div>

la qual cosa quedaria així:


Observa la repetició al CSS. El color de fons es posa a marró (brown) a diversos llocs. Per a algunes declaracions CSS, és possible declarar això més alt a la cascada i deixar que l'herència CSS resolga aquest problema de manera natural. Per a projectes no trivials, això no sempre és possible. En declarar una variable a la pseudo-classe :root, podem evitar repetició usant la variable.

:root {
  --main-bg-color: brown;
}

.un {
  color: white;
  background-color: var(--main-bg-color);
  margin: 10px;
  width: 50px;
  height: 50px;
  display: inline-block;
}

.dos {
  color: white;
  background-color: black;
  margin: 10px;
  width: 150px;
  height: 70px;
  display: inline-block;
}
.tres {
  color: white;
  background-color: var(--main-bg-color);
  margin: 10px;
  width: 75px;
}
.quatre {
  color: white;
  background-color: var(--main-bg-color);
  margin: 10px;
  width: 100px;
}

.cinc {
  background-color: var(--main-bg-color);
}

Això condueix al mateix resultat que l'exemple anterior, però permet una declaració canònica de la propietat desitjada; molt útil si volem canviar el valor a tota la pàgina més tard.

Herència de propietats personalitzades

Les propietats personalitzades hereten. Això vol dir que si no s'estableix cap valor per a una propietat personalitzada en un element donat, s'utilitza el valor del vostre element pare. Vegem el següent HTML:

<div class="un">
  <div class="dos">
    <div class="tres"></div>
    <div class="quatre"></div>
  </div>
</div>

amb el següent CSS:

.dos {
  --test: 10px;
}

.tres {
  --test: 2em;
}

En aquest cas, els resultats de var(--test) són:

  • Per a l'element class="dos": 10px
  • Per a l'element class="tres": 2em
  • Per a l'element class="quatre": 10px (heretat del pare)
  • Per a l'element class="un": valor no vàlid, que és el valor per defecte de qualsevol propietat personalitzada.

Tingueu en compte que aquestes són propietats personalitzades, no variables reals com podem trobar en altres llenguatges de programació. El valor es calcula on siga necessari, no s'emmagatzema per utilitzar-lo en altres regles. Per exemple, no es pot declarar una propietat per a un element i esperar recuperar-la a les regles del descendent d'un germà. La propietat només està configurada per al selector que coincidisca i els seus descendents, com qualsevol CSS normal.
